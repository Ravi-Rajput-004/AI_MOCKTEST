# MockPrep AI - Frontend (Client)

This directory contains the React front-end application for MockPrep.

## Tech Stack
- **Framework:** React 18, Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **State Management:** Zustand
- **Data Fetching:** TanStack Query (React Query), Axios
- **Real-time:** Socket.io-client
- **PWA:** Vanilla Service Worker & Manifest

For full documentation and setup instructions, please see the [Root README](../README.md).
